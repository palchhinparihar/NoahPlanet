# NoahPlanet

## Our Mission

NoahPlanet is a vision for a better future. We believe in empowering local communities through innovation, market insights, and sustainable practices. This platform serves as our voice – bringing together ideas for a more connected and conscious world. From local initiatives to global impact, we're building the narrative for tomorrow.

## What We Offer

**Vocal for Local** – Amplifying grassroots movements and community-driven solutions.

**Market Insights** – Understanding trends that shape our collective future.

**Future Vision** – Exploring possibilities beyond today.

**Why Noah?** – Discover what sets our approach apart.

**Core Vision** – Our fundamental principles and commitments.

Plus, a dedicated contact channel to join the conversation.

- **React 19** – Modern UI framework

- **React Scroll** – Smooth navigation experience

## Getting Started

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
